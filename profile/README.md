## Hi there 👋

**🙋‍♀️ What is this?**

Pulsar is a modern Linux distribution based on Arch Linux. Ships KDE Plasma by default with multiple desktop environments available via the Cnchi installer. Arch Linux. Nothing else.

**🌈 Contribute?**

Fork the ISO repo, tweak packages, fix Cnchi, open a PR. All GPL-2.0.

**👩‍💻 Resources**

- [pulsar-iso](https://github.com/Pulsar-Linux/pulsar-iso) — build your own ISO
- [Cnchi](https://github.com/Pulsar-Linux/cnchi) — the installer (our fork)

**🍿 Fun fact**

This project was formerly Antergos NeXT, rebranded to Pulsar per the original developer's request.

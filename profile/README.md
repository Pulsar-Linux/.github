## Hi there 👋

### 🙋‍♀️ What is this?

Pulsar Linux was an experimental Arch-based distribution created during
the early development phase of **Antergos NeXT**.

It shipped KDE Plasma by default and relied heavily on **Cnchi**, an
installer I no longer maintain.

Pulsar was never intended to become a fully separate, long-term distro.
It served as a temporary codename and testing ground before the project
was rebranded and continued as **Antergos NeXT**.

### ⚠️ Status: Discontinued

This organization is now an **archive**.

- Pulsar Linux is **no longer maintained**.
- Cnchi is **no longer maintained**.
- No updates will be made to the ISO, installer, or packages.
- This repo exists purely for **historical and archival purposes**.

If anyone intends to list Pulsar Linux (yes, SudoStation, I’m staring
directly at you), please don’t. This project is discontinued.
